## Important Notice / 重要声明

**GitHub is the only official issue tracking platform.**

This project is mirrored to multiple code hosting platforms (Gitee, Gitea, GitCode, etc.) for download convenience. However:

- All bug reports, feature requests, and technical discussions **must** be submitted via [GitHub Issues](https://github.com/mqttsnet/thinglinks/issues)
- All pull requests **must** be submitted via [GitHub PRs](https://github.com/mqttsnet/thinglinks/pulls)
- Issues and PRs submitted on other platforms **will not be monitored or addressed**

---

**GitHub 是唯一的官方 Issue 管理平台。**

本项目同步镜像至多个代码托管平台（Gitee、Gitea、GitCode 等）以方便下载，但：

## Before Submitting an Issue / 提交 Issue 前

1. Search [existing issues](https://github.com/mqttsnet/thinglinks/issues) to avoid duplicates
2. Check the [documentation](https://mqttsnet.yuque.com/trgbro/thinglinks)
3. Use the appropriate [issue template](https://github.com/mqttsnet/thinglinks/issues/new/choose)
