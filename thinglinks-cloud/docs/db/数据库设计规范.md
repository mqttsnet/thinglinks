## 📌 字符集与排序规则
- **统一字符集**: `CHARSET=utf8mb4`
- **统一排序规则**: `COLLATE=utf8mb4_general_ci`
- **说明**: 支持Emoji表情和所有Unicode字符

## 🆔 主键设计
- **禁止使用自增主键** ❌
- **使用雪花算法**:程序会自动使用雪花算法生成的唯一ID
- **字段类型**: `bigint(20)`
- **优势**: 分布式系统全局唯一，避免ID冲突

## 🏷️ 表名设计
- **模块前缀: 根据模块添加前缀，例如app_users，iot_devices，便于识别所属功能模块。**
