# 数据库初始化指南

> 本文档指导如何为 ThingLinks 平台初始化业务数据库（MySQL）和时序数据库（TDengine）。  
> 多租户架构下，每个租户拥有独立的业务库和时序库，通过 `tenantId` 进行隔离。

---

## 一、业务数据库（MySQL）

### 命名规则

```
thinglinks_base_{tenantId}
```

### 需要创建的数据库

在 Nacos 配置的 MySQL 数据源中，依次创建以下数据库：

| 数据库名称 | 用途 |
|:----------|:-----|
| `thinglinks_job` | 定时任务调度 |
| `thinglinks_ds_c_defaults` | 默认数据源（公共） |
| `thinglinks_base_1` | 租户 1 业务库 |
| `thinglinks_base_483305815051076198` | 租户 483305815051076198 业务库 |

### 操作步骤

1. 在 MySQL 中创建上述 4 个数据库。
2. 将对应的 SQL 脚本分别导入到各数据库中。

```sql
-- 示例：创建数据库
CREATE DATABASE IF NOT EXISTS thinglinks_job DEFAULT CHARACTER SET utf8mb4 COLLATE utf8mb4_general_ci;
CREATE DATABASE IF NOT EXISTS thinglinks_ds_c_defaults DEFAULT CHARACTER SET utf8mb4 COLLATE utf8mb4_general_ci;
CREATE DATABASE IF NOT EXISTS thinglinks_base_1 DEFAULT CHARACTER SET utf8mb4 COLLATE utf8mb4_general_ci;
CREATE DATABASE IF NOT EXISTS thinglinks_base_483305815051076198 DEFAULT CHARACTER SET utf8mb4 COLLATE utf8mb4_general_ci;
```

> ⚠️ 创建完成后，需将对应的初始化脚本导入到每个数据库中，否则平台无法正常启动。

## 二、时序数据库（TDengine）

### 命名规则

```
thinglinks_extend_{tenantId}
```

### 需要创建的数据库

| 数据库名称 | 用途 |
|:----------|:-----|
| `thinglinks_ds_c_defaults` | 默认时序数据源（公共） |
| `thinglinks_extend_1` | 租户 1 时序库 |
| `thinglinks_extend_483305815051076198` | 租户 483305815051076198 时序库 |

### 操作步骤

只需创建数据库即可，**无需导入额外脚本**。

依次执行以下 SQL：

```sql
-- 公共默认时序库
CREATE DATABASE IF NOT EXISTS thinglinks_ds_c_defaults
  VGROUPS 10
  BUFFER 10
  PRECISION 'ns';

-- 租户 1 时序库
CREATE DATABASE IF NOT EXISTS thinglinks_extend_1
  VGROUPS 10
  BUFFER 10
  PRECISION 'ns';

-- 租户 483305815051076198 时序库
CREATE DATABASE IF NOT EXISTS thinglinks_extend_483305815051076198
  VGROUPS 10
  BUFFER 10
  PRECISION 'ns';
```

### 关键参数说明

| 参数 | 值 | 说明 |
|:----|:---|:-----|
| `VGROUPS` | 10 | 虚拟节点组数量，影响写入并发能力 |
| `BUFFER` | 10 | 每个 vnode 的写入缓存大小（MB） |
| `PRECISION` | ns | 时间戳精度为纳秒，满足高精度采集场景 |

---

## 三、新增租户时的操作清单

当平台新增一个租户（假设 `tenantId` 为 `123456`）时，需执行以下操作：

| 步骤 | 操作 | 数据库类型 |
|:-----|:-----|:----------|
| 1 | 创建 `thinglinks_base_123456` 并导入初始化脚本 | MySQL |
| 2 | 创建 `thinglinks_extend_123456`（仅建库，无需导入脚本） | TDengine |
