## 启动

普通x86服务器：
docker run -d -p 30236:5236 --restart=always --name=dm8_test --privileged=true -e LD_LIBRARY_PATH=/opt/dmdbms/bin -e
PAGE_SIZE=16 -e EXTENT_SIZE=32 -e LOG_SIZE=1024 -e UNICODE_FLAG=1 -e INSTANCE_NAME=dm8_test -v /opt/data:
/opt/dmdbms/data dm8_single:dm8_20240715_rev232765_x86_rh6_64
mac arm服务器：
docker run -d --platform linux/amd64 -p 30236:5236 --name=dm8_test --privileged=true -e LD_LIBRARY_PATH=/opt/dmdbms/bin
-e PAGE_SIZE=16 -e EXTENT_SIZE=32 -e LOG_SIZE=1024 -e UNICODE_FLAG=1 -e INSTANCE_NAME=dm8_test -v /Users/tangyh/dm8:
/opt/dmdbms/data dm8_single:dm8_20240715_rev232765_x86_rh6_64
